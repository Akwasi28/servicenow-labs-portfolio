# ServiceNow: New User Account Creation, Group Assignment & Password Setup

## 📌 Overview
Simulated IT onboarding workflow in ServiceNow. This project demonstrates my ability to create user accounts, manage group assignments, and configure initial access within ServiceNow.

---

## 🎯 Scenario
A new employee joins the organization and requires a ServiceNow user account, appropriate group access, and an initial password setup.

---

## 🧠 Step 1: Creating the New User Account
- Navigated to "user" under organization
- Selected **User > New**
- Entered user details for *Elizabeth Jackson*
- Saved and updated the new user record
