# ServiceNow: Filtering Incidents

## 📌 Overview
Simulated IT support workflow using ServiceNow. This project demonstrates my ability to filter and analyze incident records within ServiceNow.

---

## 🎯 Scenario
Identify all incidents that are currently in progress to monitor active support tasks.

---

## 🧠 Step 1: Accessing the Incident List
- Navigated to the **All** menu
- Searched for "Service Desk"
- Selected **Incidents** to view the full incident list

*Caption: Accessing the full list of incidents.*

## 🧠 Step 2: Applying a Filter
- Clicked the filter icon
- Set condition:
  -  **Incident State**
  -  **is**
  -  **In Progress**
- Applied the filter to refine results
