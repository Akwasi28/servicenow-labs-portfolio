# ServiceNow: Creating a Visual Task Board

## 📌 Overview
Simulated IT support workflow using ServiceNow. This project demonstrates my ability to create and manage a Visual Task Board to track incident status within an ITSM environment.

---

## 🎯 Scenario
Create a Visual Task Board to monitor and organize incidents based on their current status (e.g., In Progress, On Hold, Resolved).

---

## 🧠 Step 1: Creating a Freeform Visual Task Board
- Navigated to the **All** menu
- Searched for **Self Service > Visual Task Board**
- Selected **Create Visual Task Board**
- Chose **Freeform Board**
- Named the board **"My Incidents"**
- Configured columns:
  - In Progress
  - On Hold
  - Resolved

## 🧠 Step 2: Adding Incidents to the Board
- Navigated to **Service Desk > Incidents**
- Selected multiple incident records
- Used **"Add to Visual Task Board"** option
- Assigned incidents to appropriate columns:
  - In Progress
  - On Hold
  - Resolved
