# ServiceNow Ticket Resolution Lab

## 📌 Overview
Simulated IT support workflow using ServiceNow. This project demonstrates my ability to manage, investigate, and resolve incidents within ServiceNow.

---

## 🎯 Scenario
User reports they cannot access their account after multiple attempts.

---

## 🧠 Step 1: Creating the Incident
- Under incident I selected "Create New"
- Retrieved the user’s details and completed the ticket description

## 🧠 Step 2: Assigning
- Initially I created a new user account for myself
- Assigned the ticket to myself for processing (“Akwasi A”)

*Caption: Ticket routed to the appropriate user.*

---

## 🧠 Step 3: Investigation
- Reviewed user information and error details
- Checked possible causes (permissions, system status, etc.)

---

## 🧠 Step 4: Resolution/Closing the ticket
- Unlocked the user's account 
- User was able to log in using their credentials
- Marked incident as solution provided
- Added resolution notes/date
